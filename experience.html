Front-end Development

2025 - 2026

PANDA WORDS – Chinese Learning Game Website (HSK 1-3)

  • พัฒนาเว็บไซต์เกมเพื่อการเรียนรู้ภาษาจีนระดับ HSK 1–3 โดยรับผิดชอบงานด้าน Front-end Development
  • พัฒนาเว็บไซต์ด้วย Next.js โดยใช้ JavaScript และ CSS ผ่าน Visual Studio Code พร้อมออกแบบระบบแสดงผลของเกม (Rendering Logic)
  • ออกแบบและพัฒนามินิเกมเพื่อการเรียนรู้จำนวน 4 รูปแบบ ได้แก่ เกมเติมประโยคให้ถูกต้อง, เกมเรียงประโยคให้ถูกต้อง, เกมฟังเสียงแล้วเลือกคำตอบที่ถูกต้อง และเกมสะกดคำภาษาจีนให้ถูกต้อง
  • พัฒนา Game Logic ของแต่ละเกมให้มีระบบตรวจคำตอบ และการคำนวณคะแนนที่แตกต่างกันตามประเภทของเกม
  • เชื่อมต่อระบบกับฐานข้อมูล Firebase เพื่อจัดการข้อมูลผู้เล่น และปรับเว็บไซต์ให้รองรับการใช้งานทุกอุปกรณ์ (Responsive Design)
  • ทดสอบการใช้งาน (User Testing) และปรับปรุง UX ตาม Feedback เพื่อเพิ่มประสิทธิภาพในการเรียนรู้และความสนุกของผู้เล่น
